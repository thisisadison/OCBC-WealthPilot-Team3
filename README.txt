OCBC WealthPilot — Prototype Overview

Summary:
This is a prototype for "WealthPilot" — an AI-assisted financial planning demo as part of FRANKpreneurship 2026 Team 3's project. WealthPilot lets users build an Agentic AI team of goal-focused agents (Property Planner, Retirement Planner, Growth Strategist, Income Builder) plus an Orchestrator that balances their recommendations into a single plan. The prototype includes flows for selecting agents, entering goal-specific details, assessing risk, linking accounts, and generating a recommended allocation.

Run locally:
1. cd to this folder
2. python3 -m http.server 8000
3. Open http://<MACHINE_IP>:8000/deepseek_nonsense.html on a phone/browser (same Wi‑Fi)
